## How To Use
1. Fill in ```.json``` files in ```data``` and ```metadata``` folders.
2. Run ```python3 build.py``` to build ```index.html```, ```news.html```, ```pubs.html```, and ```main.css```.
3. Open ```index.html``` to see your website!# zoekahn.github.io
